## Task App Docker

環境構築を学習するためのタスク管理アプリです。

#### 使用技術一覧

| 言語・フレームワーク | バージョン |
| -------------------- | ---------- |
| nginx                | 1.23.4     |
| Docker               | 28.0.4,    |
| Laravel              | 12.0       |
| MySQL                | 8.0        |
| Vue                  | 7.1.1      |
| Vite                 | 3.5.18     |

#### プロジェクト構成

```
task-app-docker
    └── api
        (laravelプロジェクト)
    └── client
        (Vue3プロジェクト)
    └── docker
        └── api
            └── Dockerfile
        └── client
            └── Dockerfile
        └── nginx
            └── default.conf
    └── docker-compose.yml
    └── package.json

```
